The Road to Enlightenment

Littering a dark and dreary road lay the past relics of browser-specific tags, incompatible DOMs, broken CSS support, and abandoned browsers.

We must clear the mind of the past. Web enlightenment has been achieved thanks to the tireless efforts of folk like the W3C, WaSP, and the major browser creators.

The CSS Zen Garden invites you to relax and meditate on the important lessons of the masters. Become one with the web.

So What is This About?

There is a continuing need to show the power of CSS. The Zen Garden aims to excite, inspire, and encourage participation.

Participation

You are modifying this page, so strong CSS skills are necessary. Follow the links to learn more, and use the sample files as a guide.

Benefits

Why participate? For recognition, inspiration, and a resource to show how amazing CSS really can be.

Requirements

Your CSS should validate and work across modern browsers. Use original artwork and creative themes.
